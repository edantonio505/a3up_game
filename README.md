# A3UP game

### Prerequisites
* [Python](https://www.python.org/)
* [Pygame](https://www.pygame.org/wiki/GettingStarted)



### Usage 
```
git clone https://github.com/edantonio505/a3up_game.git
cd a3up_game/
```
- Do this step if you dont have pygame installed
```
pip install pygame
```

- Run the game
```
python a3up.py
```
